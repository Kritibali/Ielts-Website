Information about results

There are no pass or fail in IELTS. You are graded on your performance in the test,using scores from 1-9 for each part of the test.

The IELTS 9-band scale

Bands
Band 9 : Expert user
Band 8 : Very Good user
Band 7 : Good user
Band 6 : Competent user
Band 5 : Modest user
Band 4 : Limited user
Band 3 : Extremely Limited user
Band 2 : Intermittent user
Band 1 : Non-user
Band 0 : Did not attempt the test
